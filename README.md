# SekawanElectro Landing Page

Landing page SvelteKit + Tailwind CSS untuk jasa service AC, kulkas, mesin cuci, bongkar pasang AC, dan elektronik rumah tangga.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output ada di folder `build/` dan siap deploy ke Cloudflare Pages.
